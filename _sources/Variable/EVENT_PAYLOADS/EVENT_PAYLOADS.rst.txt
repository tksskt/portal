===================
イベントのペイロード
===================

イベント その他のプレイヤー
--------------------------
ルール - **イベントコンテキスト** から2番目のプレイヤーペイロードを返します。

戻り値
  + プレイヤー

イベントプレイヤー
-------------------
ルール - **イベントコンテキスト** から1番目のプレイヤーペイロードを返します。

戻り値
  + プレイヤー

イベントチーム
--------------------
開催中 - **チーム** **イベントコンテキスト** から **チームId** ペイロードを返します。

戻り値
  + チームId
