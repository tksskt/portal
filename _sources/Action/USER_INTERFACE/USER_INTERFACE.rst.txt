=======================
ユーザーインターフェイス(USER INTERFACE)
=======================

--------

メッセージ(MESSAGES)
===========

すべてのカスタムメッセージをクリア(ClearAllCustomMessages)
--------------------------------
.. image:: ../../img/action/user_interface/ui1.png
与えられた **プレイヤー** または **チームId** 用のカスタムメッセージすべてを削除します。
**プレイヤー** または **チームId** が指定されていない場合、全員に向けたすべてのカスタムメッセージが削除されます。

パラメーター
  + (オプショナル)プレイヤー(Player) or チームId(TeamId)（グループ）

|
カスタムメッセージをクリア(ClearCustomMessage)
--------------------------
.. image:: ../../img/action/user_interface/ui2.png
与えられた **プレイヤー** または **チームId** に与えられている **カスタムメッセージスロットアイテム(CustomMessageSlotsItem)** からテキストを削除します。
**プレイヤー** または **チームId** が与えられていない場合、すべてのプレイヤーから **カスタムメッセージスロットアイテム(CustomMessageSlotsItem)** を削除します。

パラメーター
  + カスタムメッセージスロットアイテム(CustomMessageSlotsItem)
  + (オプショナル)プレイヤー(Player) or チームId(TeamId)（グループ）

|
カスタムメッセージを表示(DisplayCustomMessage)
-----------------------
.. image:: ../../img/action/user_interface/ui3.png
カスタムゲームモード専用の **メッセージ(Message)** を画面上に表示します。
**カスタムメッセージスロットアイテム(CustomMessageSlotsItem)** を使用することによって、この メッセージを複数の行で表示することができます。
指定された時間表示されるか、マイナスの **数値(Number)** を指定した場合には無期限に表示され続けます。
これは、**プレイヤー(Player)** または **チームId(TeamId)** によって指定されている場合を除き、全員に対して表示されます。

.. note::
  あなたには他のプレイヤーの安全で公正なゲーム体験を保証する責任があります。不快または不適切なテキストを使用して利用規約に違反した場合、アカウントがアクセス禁止になる可能性があります。

パラメーター
  + メッセージ(Message)
  + カスタムメッセージスロットアイテム(CustomMessageSlotsItem)
  + 数値(Number)（時間）
  + (オプショナル)プレイヤー(Player) or チームId(TeamId)（グループ）

|
ゲームモードメッセージを表示(DisplayGameModeMessage)
-----------------------------
.. image:: ../../img/action/user_interface/ui4.png

.. note::

パラメーター
  + メッセージ(Message)
  + (オプショナル)プレイヤー(Player) or チームId(TeamId)
